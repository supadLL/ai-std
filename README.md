## 快速开始

1. 先读 `01-3-month-learning-plan.md`，了解整体路线
2. 按计划第 1 周开始搭建环境
3. 每周日更新进度

## 学习原则

- 工作日晚上充电（读文档、写面试答案、review 代码）
- 周末输出（写新功能、调参、写文档）
- 项目 > 课程，吃透 > 吹大
- 每次面试后 10 分钟复盘
